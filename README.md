# Pixel Studio 🎨👾

**Pixel Studio** — это легковесный, современный и стильный веб-редактор для создания пиксельной графики (Pixel Art). Проект разработан на базе чистого HTML5 Canvas с использованием библиотеки jQuery. 

Идеально подходит как учебный кейс, демонстрационный проект для MakerTime или практическая работа по веб-разработке (HTML/CSS/JS).

---

## ✨ Особенности и функционал

- **🎛️ Набор инструментов:** Карандаш (Pencil), Ластик (Eraser), Заливка (Bucket Fill) и Пипетка (Color Picker).
- **🥞 Система слоев:** Добавление, переключение, дублирование, удаление и скрытие видимости слоев для сложной графики.
- **🎨 Кастомная палитра:** Выбор любого цвета через удобную встроенную палитру + набор готовых пресетов-свотчей.
- **📏 Изменение размеров:** Динамический ресайз холста под любые нужды (от 4x4 до 128x128 пикселей).
- **🎚️ Контроль холста:** Изменение фона (сетка, глубокий черный или чистый белый) и плавный зум.
- **↩️ Отмена и повтор (Undo/Redo):** Полноценная история изменений до 50 шагов назад.
- **💾 Экспорт:** Скачивание готового результата в формате `.png` с сохранением прозрачности.
- **📱 Адаптивность:** Удобные выдвижные сайдбары для комфортной работы на мобильных устройствах.

---

## 🛠️ Стек технологий

Проект построен на базовом веб-стеке без тяжелых фреймворков, что делает его легко читаемым:
1. **HTML5** (Семантическая разметка + элемент `<canvas>`)
2. **CSS3** (Кастомные свойства/переменные, современный темный интерфейс, Flexbox и CSS Grid)
3. **jQuery (v3.7.1)** (Быстрая и лаконичная работа с DOM-деревом и событиями)
5. **SVG в коде (Data URL)** (Для мгновенной загрузки векторных иконок интерфейса и фавиконки сайта)

## 🚀 Как запустить

Проект полностью автономен и работает прямо в браузере без необходимости поднимать локальный сервер или устанавливать зависимости через `npm`.

1. Клонируйте репозиторий или скачайте файлы проекта.
2. Убедитесь, что все три основных файла находятся в одной папке:
   - `main.html` — основная разметка
   - `css.css` — таблицы стилей
   - `js.js` — логика приложения
3. Просто дважды кликните по файлу **`main.html`**, чтобы открыть Pixel Studio в любом современном браузере.

## 🧠 Горячие клавиши (Shortcuts)

Для ускорения процесса рисования настроены следующие хоткеи:
- `P` — Инструмент **Карандаш**
- `E` — Инструмент **Ластик**
- `G` — Инструмент **Заливка**
- `I` — Инструмент **Пипетка**
- `Ctrl + Z` — **Отмена** последнего действия
- `Ctrl + Y` — **Повтор** отмененного действия

## 📝 Настройка под себя

- **Размер холста по умолчанию:** Меняется в файле `js.js` в объекте `const editor` (свойства `width` и `height`), а также в `main.html` в атрибутах `value` у инпутов размера холста.
- **Кастомные цвета темы:** Все цвета интерфейса вынесены в начало файла `css.css` в блок `:root`. Вы можете легко перекрасить приложение, просто заменив значения переменных (например, `--primary` для главного акцентного цвета).
