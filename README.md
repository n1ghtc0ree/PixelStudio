# Pixel Studio 🎨👾

**Pixel Studio** — это легковесный, современный и стильный веб-редактор для создания пиксельной графики (Pixel Art). Проект разработан на базе чистого HTML5 Canvas с использованием библиотеки jQuery.

Интерфейс полностью адаптирован как для работы на ПК, так и на мобильных устройствах (смартфонах и планшетах) с сенсорными экранами. Идеально подходит в качестве демонстрационного проекта для MakerTime или практической работы по веб-разработке.

---

## ✨ Особенности и функционал

- **🎛️ Набор инструментов:** Карандаш (Pencil), Ластик (Eraser), Заливка (Bucket Fill) и Пипетка (Color Picker).
- **🥞 Система слоев:** Добавление, переключение, дублирование, удаление и скрытие видимости слоев для создания сложных артов.
- **🎨 Кастомная палитра:** Выбор любого цвета через встроенную палитру + компактная сетка готовых свотчей (автоматически перестраивается под мобильные экраны).
- **📏 Изменение размеров:** Динамический ресайз холста под любые нужды (от 4x4 до 128x128 пикселей).
- **🎚️ Контроль холста:** Переключение подложки рабочей области (классическая pixel-art шахматка, глубокий черный или чистый белый фон).
- **↩️ Отмена и повтор (Undo/Redo):** Полноценная история изменений до 50 шагов назад.
- **💾 Экспорт:** Скачивание готового результата в формате `.png` с сохранением прозрачности.
- **📱 Ультра-адаптивность и Touch-поддержка:** - Сайдбары на смартфонах открываются во весь экран и имеют кнопку-крестик для удобного закрытия.
  - Системные кнопки шапки на узких экранах автоматически прячутся в выдвижное меню, чтобы верстка не ломалась.
  - Полноценно реализовано непрерывное рисование линиями на сенсорных экранах (пальцем или стилусом) с блокировкой случайных жестов браузера.

## 🛠️ Стек технологий

Проект построен на чистом веб-стеке без тяжелых фреймворков и лишних зависимостей:
1. **HTML5** (Семантическая разметка + элемент `<canvas>`)
2. **CSS3** (Медиа-запросы для узких дисплеев, кастомные переменные, темная тема, Flexbox и CSS Grid)
3. **jQuery (v3.7.1)** (Лаконичная работа с DOM-деревом, обработка мышиных и сенсорных `touch`-событий)
4. **SVG в коде (Data URL)** (Все иконки интерфейса и фавиконка вкладки зашиты прямо в код для мгновенной загрузки)

## 🚀 Как запустить

Проект полностью автономен и работает прямо в браузере.

1. Убедитесь, что все три основных файла находятся в одной папке:
   - `main.html` — основная разметка
   - `css.css` — таблицы стилей
   - `js.js` — логика приложения
2. Просто дважды кликните по файлу **`main.html`**, чтобы открыть Pixel Studio.

## 🧠 Горячие клавиши (Shortcuts)

Для ПК-версии настроены следующие хоткеи:
- `P` — Инструмент **Карандаш**
- `E` — Инструмент **Ластик**
- `G` — Инструмент **Заливка**
- `I` — Инструмент **Пипетка**
- `Ctrl + Z` — **Отмена** действия
- `Ctrl + Y` — **Повтор** действия

## 📝 Настройка под себя

- **Размер холста по умолчанию:** Настраивается в `js.js` в начале объекта `const editor` (`width: 64`, `height: 64`), а также в файле `main.html` в атрибутах `value` у инпутов размера.
- **Кастомные цвета темы:** Все цвета интерфейса вынесены в начало файла `css.css` в блок `:root`. Можно легко изменить главный акцентный цвет, поправив переменную `--primary`.
